MeshJS provides a comprehensive set of utility functions for working with Cardano data. Use these utilities to serialize scripts into addresses, deserialize on-chain data, resolve hashes, and work with smart contract blueprints.

## Overview

| Utility | Purpose |
|---------|---------|
| [Blueprints](/apis/utilities/blueprints) | Create and manage smart contract blueprints for spending, minting, and withdrawal validators |
| [Serializers](/apis/utilities/serializers) | Encode scripts and data into CBOR or bech32 format |
| [Deserializers](/apis/utilities/deserializers) | Parse CBOR or bech32 encoded data into usable objects |
| [Resolvers](/apis/utilities/resolvers) | Convert between different Cardano data formats (hashes, addresses, epochs) |

## Quick Start

Install the MeshJS SDK to access all utility functions:

```bash
npm install @meshsdk/core
```

Import the utilities you need:

```tsx
import {
  serializePlutusScript,
  deserializeAddress,
  resolveDataHash,
  SpendingBlueprint,
} from "@meshsdk/core";
```
